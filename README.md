# Sales Register

## Description
This project is a simple Python program that allows recording the sales of a store during the day.  
The user can enter products, quantities, and prices, and the program calculates the total money collected.

## Objective
To facilitate the recording of sales in a small store and obtain a clear summary at the end of the day.

## Features
- Register the name of a product
- Enter the quantity sold
- Enter the price of the product
- Calculate the total of each sale
- Show the accumulated total of all sales

## How it works
1. The program asks for the name of the product.
2. Then it asks for the quantity sold.
3. Next, it asks for the product price.
4. It calculates the total of the sale.
5. It repeats the process until all sales are recorded.
6. Finally, it shows the total money collected **until the user decides not to add more products**.
